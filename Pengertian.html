Dasar CSS

Properti adalah perintah yang kita berikan untuk mengatur tampilan elemen HTML. Misalnya mengatur warna, ukuran, jenis huruf, atau jarak.

Sedangkan value adalah spesifikasi nilai yang kita isi ke dalam properti tersebut. Ini yang menentukan hasil akhir tampilannya seperti apa.

Contoh Struktur:

background-color:redProperti = Value
font-size:20pxProperti = Value
